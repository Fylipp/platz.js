placeholder.js

Painless placeholders for lists and list-like elements

Installation

Getting placeholder.js is very simple. Download the latest build and link the minified files in your HTML file's head section. 

<link rel="stylesheet" href="placeholder.min.css">
<script src="placeholder.min.js"></script>

Usage

In order to use placeholder.js all lists and list-like elements that should receive placeholder.js functionality have to be registered. This can be done with placeholderjs_init() which automatically finds all elements in the DOM that use placeholder.js attributes. Below you can see how to use this function in your code. Note that calling this function multiple times can have unwanted side effects.

With jQuery

$(document).ready(function () {
    placeholderjs_init();
});

Without jQuery

window.onload = function () {
    placeholderjs_init();
};

If lists and list-like elements are added on-the-fly you must call placeholderjs_register(newList). Below you can see how to use this function. Note that the jQuery also works for selectors that match multiple elements.

With jQuery

var element = $('#my-list');
element.each(function() {
    placeholderjs_register(this);
});

Without jQuery

var element = document.getElementById('my-list');
placeholderjs_register(element);

Primitive Placeholder

This is the simplest placeholder. It is displayed as text in the content of the ::after or ::before element.

Below ::after is used. In order to have the placeholder in ::before you simply have to use data-placeholderjs-before. 

<ul data-placeholderjs-after="The list is empty">
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Advanced Placeholder

Sometimes a placeholder element should visually replace the list. Below you can see this feature being used with MaterializeCSS colllections. Had the element not been replaced the list border would be visible. Any HTML element can be used as a placeholder this way.
